# 概要

* このページで動作する、html/css/js製のツール
* 外部APIなどは使用しない

# 要件

* タイマーのスタート・ストップが1ボタンで出来る
* タイマースタート・ストップを1つのレコードとして記録する
  * 記録項目
    * 開始日時
    * 終了日時
  * 記録方法 : URLパラメータ
    * タイムスタンプをCSV形式で連結させるだけとする (JSON化不要、URIエンコード不要)
* 指定時間経過時にアラームを鳴らす
  * hour/min/secをそれぞれ指定できるようにする
  * アラーム時「タイマーは止めず、アラームだけ止める」ためのボタンをポップアップ表示
  * アラーム音は、柔らかく温かみのある木琴のような音
    * ユーザをびっくりさせないよう配慮してほしい
  * 技術的な注意 : アラーム時間の判定は「経過時間 > 設定時間」ではなく「経過時間 == 設定時間」でよい (setIntervalが1秒未満であれば、秒の単位まで `==` で比較すれば十分な精度でアラームを鳴らすことができる)
* レコードをtable形式で表示する
  * カラム
    * 開始日時
    * 終了日時
    * 経過時間 (開始日時と終了日時から計算する)
* 「経過時間」の総計を表示する
* すべてをリセットするボタンを提供する
  * 確認ダイアログ (confirmなど) をはさむ
  * 目的 : これが押されるまで、ページをリロードされても、URLパラメータで復元できるようにする
* かわいいデザイン
  * 「ちいかわ」等のゆるかわ系
  * 視認性を意識 (パステルカラーばかりに偏重しない)
  * 無駄なアニメーションは不要
